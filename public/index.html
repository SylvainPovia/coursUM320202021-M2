/state/:state.:format?

Returns a json or xml+rdf of the covid-19 informations for each state and return the cities in the state. You must specify the format you want, .xml download an xml+rdf file and .json send you a json file.

/state/:state/city/:city.:format?

Returns a json or xml+rdf of the covid informations for the state of the city and return it demographic informations. The same here, you must specify in which format you want the data.

/rdf_vocabulary

Returns an xml+rdf that contains the api data descriptions.
